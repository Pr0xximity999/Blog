> I want to start by talking about the history of this website.

It all started when a friend of mine showed me a website he found. The website in question being dimden's website. The website in question^ This page really inspired me to make my own. So i decided to make my own.
I looked for some hosting options and discovered that you can host a github repository. In the settings of a repository you have this button to lead to your pages settings In this page are a whole lot of settings with which you can set up the website. Now this webpage normally has a set domain name, something like [github_username].pages.github.io or something. Yeah i didnt want that. So i paid for a custom domain, the one in the url bar right now, tominhisroom.dev

>A blank slate

Of course this website didnt start out the way you see it here. It started out as this. A blank slate Nothing, not even some text. So i did just that and added something :3

>After this i tried to make the url actually work, which took like 2 days to do since i screwed myself over by using a .dev domain..

Most domains can be connected with via both HTTP and HTTPS. With .dev sites you can only connect over HTTPS. Now you might be wondering, so why is this annoying? Well to explain it shortly, the difference between the two is that the S in https stands for secure It just means you need a certain certificate for your url to work. Which for someone like me who never really did anything web-related was some work to figure out. And to get one was quite the hassle before i discovered that GitHub can do it automatically for you. So i wasted 2 days of work for nothing :) After that the one and only, fabled and feared.. sex button was added When you clicked it, it would close the website, was very funny.
I also added an image which my friend asked to add And then some more images.. Added some styling in the form of a crt effect and colored boxes also a description (sex button got removed because capitalism) And finally, some text styling, a TODO list, a blog section, and the famous noita poop stone And that's where i am right now, first page done ^w^ hopefully many to come
